การติดตั้ง Docker

Docker คือแพลตฟอร์มสำหรับพัฒนา, จัดส่ง, และรันแอปพลิเคชันในสิ่งที่เรียกว่า "Containers" ซึ่งช่วยให้เราสามารถแยกแอปพลิเคชันออกจากสภาพแวดล้อมและทำให้มั่นใจได้ว่ามันจะทำงานเหมือนกันในทุกที่

ขั้นตอนการติดตั้งบน Ubuntu

การติดตั้ง Docker บนระบบปฏิบัติการ Ubuntu สามารถทำได้ง่ายๆ ผ่าน Command Line ดังนี้

  1. Uninstall เวอร์ชั่นเก่าออก 
    for pkg in docker.io docker-doc docker-compose docker-compose-v2 podman-docker containerd runc; do sudo apt-get remove $pkg; done
  2. อัปเดต Package Index ติดตั้ง Package ที่จำเป็น และ เพิ่ม Docker’s official GPG key: 
    sudo apt-get update
sudo apt-get install ca-certificates curl gnupg lsb-release
sudo install -m 0755 -d /etc/apt/keyrings
sudo curl -fsSL https://download.docker.com/linux/ubuntu/gpg | sudo gpg --dearmor -o /etc/apt/keyrings/docker.gpg
sudo chmod a+r /etc/apt/keyrings/docker.gpg
  3. ตั้งค่า Repository: 
    echo \
  "deb [arch=$(dpkg --print-architecture) signed-by=/etc/apt/keyrings/docker.gpg] https://download.docker.com/linux/ubuntu \
  $(lsb_release -cs) stable" | sudo tee /etc/apt/sources.list.d/docker.list > /dev/null
  4. ติดตั้ง Docker Engine: 
    sudo apt-get update
sudo apt-get install docker-ce docker-ce-cli containerd.io docker-buildx-plugin docker-compose-plugin -y
  5. ตรวจสอบการทำงาน Service Docker และให้ทำงานอัตโนมัติทุกครั้งที่ Reboot
    sudo systemctl status docker
sudo systemctl enable docker
  6. ทดสอบการติดตั้ง:
    รัน image `hello-world` เพื่อตรวจสอบว่า Docker ทำงานถูกต้อง 
    sudo docker run hello-world

ภาพตัวอย่างผลลัพธ์จากการรัน `hello-world`

การจัดการและทวงคืนพื้นที่ (Disk Space) จาก Docker

คำสั่งสำหรับเคลียร์และลบ Container, Image, Network, และ Volume ที่ไม่ได้ใช้งานเพื่อทวงคืนพื้นที่ในเครื่อง

  1. ลบคอนเทนเนอร์ที่หยุดทำงานแล้วทั้งหมด 
    docker container prune
  2. ลบ Image ที่ไม่มี Container ใดใช้งาน 
    docker image prune -a
  3. ลบ Network ที่ไม่ได้ใช้ 
    docker network prune
  4. ลบ Volume ที่ไม่ได้ใช้ (ข้อควรระวัง: ข้อมูลถาวรจะถูกลบ) 
    docker volume prune
  5. ลบ Build Cache ที่ไม่ได้ใช้งาน 
    docker builder prune
  6. ลบทุกอย่างที่ไม่ได้ใช้งานในคำสั่งเดียว (ยกเว้น Volume) 
    docker system prune -a
  7. ลบทุกอย่างที่ไม่ได้ใช้งานทั้งหมด (รวม Volume - ระวังข้อมูลหาย) 
    docker system prune -a --volumes
