ทักษะการพัฒนาระบบประเมินบุคลากรด้วยระบบเทคโนโลยีสารสนเทศสมัยใหม่ (ปวช.)

หน้านี้สรุปคุณลักษณะขั้นต่ำของอุปกรณ์ (Hardware) และซอฟต์แวร์ (Software) ที่จำเป็นต้องใช้ในการแข่งขันทักษะการพัฒนาระบบประเมินบุคลากรด้วยระบบเทคโนโลยีสารสนเทศสมัยใหม่ ระดับ ปวช. โดยอ้างอิงจากเกณฑ์การแข่งขัน ปีการศึกษา 2568 - 2572

สเปคเครื่อง สำหรับพัฒนาระบบ ที่แนะนำ (Computer Specifications)

เพื่อให้การแข่งขันดำเนินการ อย่างราบรื่น เครื่องคอมพิวเตอร์ที่ใช้ในการพัฒนา ของคุณควรมีสเปคขั้นต่ำและสเปคที่แนะนำดังนี้:

ทรัพยากร ขั้นต่ำ (พอใช้งานได้) แนะนำ (สำหรับใช้งานจริง)
CPU 4 Cores 8 Cores ขึ้นไป
RAM 4 GB 8 GB - 16 GB ขึ้นไป
Storage 250 GB (SSD) 500 GB+ (แนะนำ NVMe/SSD)
OS Windows 11 / Ubuntu Desktop 24

สเปคเครื่อง Server กรรมการ ที่แนะนำ (Server Specifications)

เพื่อให้ทุก Service ที่เราจะติดตั้งทำงานได้อย่างราบรื่น Server ของคุณควรมีสเปคขั้นต่ำและสเปคที่แนะนำดังนี้

ทรัพยากร ขั้นต่ำ (พอใช้งานได้) แนะนำ (สำหรับใช้งานจริง)
CPU 8 Cores 12 Cores ขึ้นไป
RAM 8 GB 16 GB ขึ้นไป
Storage 250 GB (SSD) 500 GB+ (แนะนำ NVMe/SSD)
OS Ubuntu Server 24.04 LTS (หรือเวอร์ชัน LTS ล่าสุด)

*LTS (Long-Term Support) คือเวอร์ชันที่ได้รับการสนับสนุนด้านความปลอดภัยในระยะยาว ทำให้เหมาะกับงาน Server

1. สิ่งที่ผู้เข้าแข่งขันต้องเตรียม

ผู้เข้าแข่งขันแต่ละทีมจะต้องนำอุปกรณ์ดังต่อไปนี้มาเองในวันแข่งขัน.

อุปกรณ์ จำนวน/รายละเอียด
เครื่องคอมพิวเตอร์ หรือ โน้ตบุ๊ก 1 เครื่องต่อทีม
จอภาพเพิ่มเติม (External Monitor) 1 จอภาพต่อทีม
ไดรเวอร์ (Driver) ไดรเวอร์ที่จำเป็นสำหรับเครื่องคอมพิวเตอร์และอุปกรณ์ที่นำมา

หมายเหตุ: อุปกรณ์นี้ต้องส่งในวันแรกของการลงทะเบียน คณะกรรมการดำเนินการต้องลบข้อมูลในเครื่องผู้เข้าแข่งขัน และให้ผู้เข้าแข่งขันติดตั้งโปรแกรมใหม่ทั้งหมดตามเกณฑ์ที่กำหนด.
