ตัวอย่าง: ระบบประเมินครู (PES - Personnel Evaluation System)

นี่คือโปรเจกต์ตัวอย่างเพื่อสาธิตการนำสภาพแวดล้อม DevOps ที่เราได้ตั้งค่ามาทั้งหมดไปใช้งานจริง โปรเจกต์ "ระบบประเมินครู" (PES) เป็นเว็บแอปพลิเคชันที่พัฒนาขึ้นโดยแบ่งส่วน Backend และ Frontend ออกจากกัน และใช้ Docker Compose ในการรัน Service ทั้งหมด

คุณสามารถดาวน์โหลดซอร์สโค้ดของโปรเจกต์เพื่อนำไปศึกษาและทดสอบบนเครื่องของคุณเองได้

ดาวน์โหลดไฟล์ PES.zip

    ตัวอย่างโปรเจคอื่น ๆ

  • ตัวอย่าง Project demo-task : ตัวอย่างสำหรับนำไปทดลองใช้งานจริง ดาวน์โหลดที่นี่ (ไฟล์ zip ข้างในมี 3 ส่วนได้แก่ Database, Backend, Frontend)
  • ตัวอย่าง Project skill_test_2568_v1 : ตัวอย่างสำหรับนำไปพัฒนาต่อ ระดับ ปวช. ดาวน์โหลดที่นี่ หรือ Github
  • ตัวอย่าง .env ที่ใช้กับ skill_test_2568 ตัวอย่างไฟล์ .env สำหรับนำไปพัฒนาต่อ ระดับ ปวช. ดาวน์โหลดที่นี่
  • ตัวอย่าง Database : ตัวอย่าง Database สำหรับนำไปพัฒนาต่อ ระดับ ปวช. ดาวน์โหลดที่นี่
  • ตัวอย่าง Project skill_test_2568_v2 : ตัวอย่างสำหรับนำไปพัฒนาต่อ ระดับ ปวส. ดาวน์โหลดที่นี่ หรือ Github

วิธีการนำไปทดสอบ

หลังจากดาวน์โหลดและเตรียมไฟล์เรียบร้อยแล้ว ให้ทำตามขั้นตอนต่อไปนี้เพื่อรันโปรเจกต์

1. Unzip ไฟล์และตรวจสอบโครงสร้าง

แตกไฟล์ `PES.zip` ไปยังตำแหน่งที่คุณต้องการ ในตัวอย่างนี้จะใช้ `C:\PES` โครงสร้างภายในโฟลเดอร์ควรมีลักษณะดังนี้: 

C:\PES
├── backend/
├── frontend/
├── .env
├── .gitignore
├── docker-compose.yml
└── schema.sql

2. ติดตั้ง Docker

ตรวจสอบให้แน่ใจว่าคุณได้ติดตั้ง Docker บนเครื่องคอมพิวเตอร์ของคุณเรียบร้อยแล้ว เช่น Docker Desktop สำหรับ Windows หรือ Mac

3. รันโปรเจกต์ด้วย Docker Compose

เปิดโปรแกรม Command Prompt (CMD) หรือ PowerShell ขึ้นมา แล้วรันคำสั่งต่อไปนี้ทีละบรรทัด 

c:
cd \
cd PES
docker compose up -d --build

รอสักครู่เพื่อให้ Docker ทำการดาวน์โหลด Image, สร้าง (Build) และเริ่มการทำงานของ Container ทั้งหมด

4. เข้าสู่ระบบเพื่อทดสอบ

เมื่อ Container ทำงานเรียบร้อยแล้ว คุณสามารถเข้าถึงส่วนต่างๆ ของระบบได้ผ่านเว็บเบราว์เซอร์

  • หน้าเว็บหลัก (Frontend): http://localhost:3000
  • หน้าจัดการฐานข้อมูล (Adminer): http://localhost:8080

ใช้ข้อมูลต่อไปนี้เพื่อล็อกอินเข้าสู่ระบบ:

  • Username: admin@ccollege.ac.th
  • Password: 12345678
